          在 MS-Windows 上修改键盘映射

1. 打开注册表: Win-R regedit RET

2. 导航至:
   “计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\
    Keyboard Layout”

3. 单击“Keyboard Layou”, 在右侧窗格中, 右击空白区域, 选择“新建”->“二进
   制值”.

4. 将新建的“二进制值”命名为“Scancode Map”.

5. 双击“Scancode Map”, 编辑“数值数据”框, 编辑完成之后点击确定.

6. 重启.

----------------------------------------

          “数值数据”的格式

举例: (按下 NumberLock 键   -> 相当于按下 “`” backquote;
       按下 CapitalsLock 键 -> 啥也不干)
00 00 00 00  00 00 00 00
03 00 00 00  29 00 45 00
00 00 3A 00  00 00 00 00

1. 前 8 个字节 是 固定格式.

2. 最后 4 个字节 是 固定格式, (据 ChatGPT 说) 表示 null 映射.

3. 第 9~12 这 4 个字节 组成 小端格式 的 整数, 表示 映射数目 (包含 末尾
   的 null 映射).
   这里设置了 2 个映射, 所以 该值 为 2 + 1 = 3.

4. NumberLock 的 扫描码 是 0x45, “`” 的 扫描码 是 0x29, 所以上例中
   “29 00  45 00” 表示 “`” 还可来自 扫描码为 0x45 的键击;

   CapitalsLock 的 扫描码 是 0x3A, 将其发出的 键 的 扫描码 置零 即可.
